## ğŸ¨ Customization Made Easy

### Change Colors
Edit `_variables.yml`:
```yaml
colors:
  primary: "#YourColor"
  secondary: "#YourColor"
  # ... more
```

### Change Fonts
Edit `_variables.yml`:
```yaml
typography:
  heading: "'Your Font', sans-serif"
  body: "'Your Font', sans-serif"
```

### Add/Remove Pages
Edit `_quarto.yml`:
```yaml
navbar:
  left:
    - text: "New Page"
      href: new-page.qmd
```

### Customize Components
Edit `assets/css/components.css` or create new classes.
